# TowzinSaravan-Management-System
Towzin Saravan Management System

A multi-layer C# and SQL Server enterprise application developed for internal operations at Siloo Saravan (Tala Samar-e Moghan Co.).
This system was designed to streamline weighing workflows, improve data accuracy, and automate daily business processes.

Key Features

Multi-layer architecture (App, Business, DataLayer, Utility, ViewModel)

SQL Server–based data management

Centralized weighbridge (towing) operations

Automated data validation and workflow steps

User management & authentication

Reporting and analytics modules

Optimized for internal operations and team efficiency

Technologies Used

C# .NET Framework / Visual Studio

SQL Server

Entity Framework (if used)

Layered Architecture (N-tier)

Git Version Control

Project Structure

TowzinSaravan.App – Main application UI

TowzinSaravan.Business – Business logic layer

TowzinSaravan.DataLayer – Database operations

TowzinSaravan.Utility – Common helpers and utilities

TowzinSaravan.ViewModel – Data models for UI binding

Background

This project was developed during my time working as IT Manager at Siloo Saravan, where I was responsible for:

Managing internal IT systems

Designing and implementing new software tools

Improving data accuracy in daily weighbridge operations

Leading automation and digitalization efforts

The project was left incomplete due to organizational changes, but the structure and logic demonstrate enterprise-level software development skills.
